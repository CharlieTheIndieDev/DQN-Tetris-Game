AI INSTRUCTIONS

executables are present in the folder labelled executables
there are two folders run_train and run_eval


1. To train: 
Go to run_train -> run_train.exe

It will take some time but will execute
---------------------------------------
To execute tensorboard after train command is:

tensorboard --logdir=logs/ --host localhost --port 8088

---------------------------------------

2. To run the evaluation:
Go to run_eval ->run_eval.exe

You should now see console outputs and the ai playing the tetris game

-----------------------------------------------------

*Note
In case you would like to compile and run a directory called Anaconda environment is provided

In it you should find the spec-file.txt and the virtual environment yml file.
The install instructions in that folder will assist in setting up the environment in your local machine
